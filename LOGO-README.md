# Logo Kelurahan Rawajati

## Deskripsi

Logo custom Kelurahan Rawajati dibuat dengan format SVG (Scalable Vector Graphics) yang profesional dan modern.

## Filosofi Desain

### Warna
- **Merah (#E30613)**: Melambangkan keberanian dan semangat pelayanan
- **Tosca (#00A651)**: Melambangkan kemakmuran dan pertumbuhan
- **Emas (#FFD700)**: Melambangkan keunggulan dan prestise

### Elemen Visual

1. **Monas Stylized**: Ikon Jakarta yang merepresentasikan identitas DKI Jakarta
2. **Perisai (Shield)**: Melambangkan perlindungan dan pelayanan kepada masyarakat
3. **Bintang**: Melambangkan keunggulan dan cita-cita luhur
4. **Padi (17 butir)**: Melambangkan kemakmuran dan tanggal kemerdekaan (17 Agustus)
5. **Lingkaran**: Melambangkan kesatuan dan persatuan

## Lokasi File

```
assets/images/logo-rawajati.svg
```

## Penggunaan

Logo sudah terintegrasi di semua halaman HTML:

```html
<a href="index.html" class="nav-logo">
    <img src="assets/images/logo-rawajati.svg" alt="Logo Kelurahan Rawajati" class="logo-img">
    <span>Kelurahan Rawajati</span>
</a>
```

## Ukuran

- **Display Size**: 60x60 pixels (di navbar)
- **SVG ViewBox**: 400x400 units
- **Format**: SVG (scalable tanpa kehilangan kualitas)

## Keunggulan Logo SVG

1. **Scalable**: Dapat diperbesar tanpa kehilangan kualitas
2. **Ringan**: Ukuran file kecil (< 5KB)
3. **Responsive**: Menyesuaikan dengan berbagai ukuran layar
4. **Professional**: Tampilan tajam di semua device
5. **Custom**: Dirancang khusus untuk Kelurahan Rawajati
